# Loan Approval Prediction

## Description
This project predicts whether a loan application will be approved using Machine Learning classification algorithms.

## Machine Learning Algorithms Used
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Gaussian Naive Bayes

## Libraries Used
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Dataset
loan_approval_data.csv

## Workflow
1. Data Cleaning
2. Handling Missing Values
3. Feature Engineering
4. Label Encoding
5. One-Hot Encoding
6. Feature Scaling
7. Model Training
8. Model Evaluation
